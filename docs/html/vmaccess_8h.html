§ vmem_init()

void vmem_init ( void  )

Connect to virtual memory.

Will request shared memory and then map it to a vmem_struct. Assumes that the shared memory already exists and has been initialized.

Connect to virtual memory.

Will request and truncate shared memory and then map it to a vmem_struct. The administration struct (with the semaphore) and page table in vmem_struct will then be initialized.
