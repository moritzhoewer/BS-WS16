§ load_page()

void load_page ( int  page,
int  frame 
)

Loads a page from disk.

SIDEEFFECT: Will change (overwrite) the data associated with frame.

Precondition: page is between 0 and VMEM_NPAGES frame is between 0 and VMEM_NFRAMES

Postcondition: data stored in frame will be overwritten

Parameters
[in]pagethe page to load
[in]framethe frame to load into

§ vmem_init()

void vmem_init ( void  )

Initialize virtual memory.

Will request and truncate shared memory and then map it to a vmem_struct. The administration struct (with the semaphore) and page table in vmem_struct will then be initialized.
