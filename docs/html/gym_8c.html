§ gym_get_weights()

void gym_get_weights ( int  total,
int  weight_counts[] 
)

[MONITOR METHOD] Hands out weights to the caller.

This method will check which weights are available and then try to find a combination that sums up to the required total. If it finds one, the weights will be handed out (signaled by a return value of 0). If not, an error code is returned

Parameters
[in]totalthe desired total weight
[out]weight_countswill contain the weights if method returned 0
